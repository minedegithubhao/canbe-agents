# 第一版

评估的根本目的：**把“我感觉“，变为同一套题、同一套规则下，指标的变化”。**

主要解决的问题

```text
1. 该找的资料找到了吗？
2. 找到的资料排得靠前、够干净吗？
3. 最终答案有没有基于资料正确回答，且没有越界乱说？
```

第一阶段指标：

指标

| 指标                      | 看什么问题                          |
| :------------------------ | :---------------------------------- |
| hit_at_k                  | 有没有至少找对一个 chunk            |
| context_recall_at_k       | 标准证据找全了多少                  |
| mrr_at_k                  | 正确 chunk 排得靠不靠前             |
| precision_at_configured_k | 固定 topK 策略下正确比例如何        |
| precision_at_effective_k  | 实际送给 LLM 的上下文干不干净       |
| avg_effective_k           | 阈值过滤后平均还剩多少上下文        |
| zero_context_rate         | 阈值是否过严，导致无上下文/fallback |

目标是先验证 RAG 的基础闭环：

```text
问题进来 -> 是否召回正确 FAQ -> 正确 FAQ 排名是否靠前
```

### 参数说明

## eval_type

表示“这道评估题主要考系统哪种能力”。

| 取值               | 含义                                                         | 第一阶段是否启用 |
| :----------------- | :----------------------------------------------------------- | :--------------- |
| single_faq         | 单 FAQ 命中题。一个问题只对应一条标准 FAQ，用来测试基础召回能力 | 启用             |
| colloquial_rewrite | 口语化改写题。测试用户日常表达下是否还能召回正确 FAQ         | 启用             |
| typo_or_alias      | 错别字/简称/别名题。测试输入不规范时的检索鲁棒性             | 启用             |
| multi_faq          | 多 FAQ 综合题。一个问题需要召回多条 FAQ                      |                  |
|                    |                                                              |                  |

## difficulty

| 取值   | 含义                                             |
| :----- | :----------------------------------------------- |
| easy   | 和原 FAQ 问法接近，关键词明显，目标 FAQ 容易召回 |
| medium | 有口语化、同义词、省略表达，需要一定语义召回能力 |
| hard   | 有错别字、别名、模糊表达，容易召错或排错         |

默认比例：

```
{  "easy": 0.5,  "medium": 0.35,  "hard": 0.15 }
```

## question_style

表示“测试问题是怎么从源 FAQ 问题变来的”。

| 取值                | 含义                   |
| :------------------ | :--------------------- |
| original            | 原始 FAQ 问法          |
| semantic_equivalent | 语义等价改写           |
| colloquial          | 口语化表达             |
| keyword_query       | 关键词式搜索           |
| typo                | 错别字、漏字、输入错误 |
| alias               | 简称、别名、同义替换   |
